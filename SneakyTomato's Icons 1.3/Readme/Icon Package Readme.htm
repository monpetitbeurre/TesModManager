Introduction

These icons are compatible with Vista (256x256) and XP (64x64)

They were designed for Vista, because the differences in the many versions I included are seen best in Vista.

They are however fully compatible with XP.

Please post your ideas on how this package may become better!

Enjoy!

Installation

 

To apply the icons, right click on the Oblivion shortcut, and in the place of the standard icon select the new icon included in this zip file.
